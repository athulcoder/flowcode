# FlowCode

FlowCode is a simple local AI coding assistant built using Ollama and DeepSeek-Coder.

It helps you generate code, build projects, and understand programming concepts — all running locally on your machine.

## Features

- ChatGPT-like interface
- Runs completely locally
- Powered by DeepSeek-Coder
- Clean UI with code highlighting
- Copy-ready code blocks

## Tech Stack

- Next.js
- Tailwind CSS
- Ollama
- DeepSeek-Coder

```bash
git clone https://github.com/your-username/flowcode.git
